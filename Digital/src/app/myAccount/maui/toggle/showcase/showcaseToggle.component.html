A toggle element that can be used in forms:

  • To allow users to enable or disable an online service (eBill/energy insights).
  • To enable a True/False or On/Off selection

A loading spinner can be displayed if the action or request takes time.

Attributes
toggleValue boolean Sets the toggle value to 'On' or 'Off'
isLoading boolean Determines the loading state of the toggle
isDisabled boolean Makes the toggle unclickable and displays a cursor style of 'not-allowed'
