The Icon List component is a reusable component that displays a list of items with icons. It is to be used on confirmation screens to provide customers with a summary of confirmation details or suggestions for other tasks to complete in my account. It is to be placed in the UI container. Refer to MAUI Component Library for details. Note the component examples below have not been inlcuded in a container to illustrate the borders of the component.
