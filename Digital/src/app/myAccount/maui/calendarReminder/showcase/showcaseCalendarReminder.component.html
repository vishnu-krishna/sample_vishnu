The MAUI Calendar reminder component can be used in 2 ways:
  • A one off reminder event
  • A recurring reminder event

The definition of the CalendarReminderModel that is passed in defines the type

If a frequency, interval and end date are passed as part of the model then the reminder will be recurring

All other attributes are common to both

Note: While Yahoo is included is the third party service (Addevent), Yahoo does not support recurring events

so it was decided to remove that option altogether.

The component returns the link from the selected option for use with tracking for example

Attributes
icsFileName stringDefaults to 'AGL bill reminder'
timezone stringDefaults to 'Australia/Sydney'
calendarReminderModel CalendarReminderModel
CalendarReminderModel Attributes
title stringTitle of the event
start stringThis is the start date of the event (format YYYY/MM/DD start time is '%208:00' ie 8am)
calendarType enum / stringThis is set within the component eg Apple, Google, Outlook, OutlookCom
description stringThis is the body of the reminder event
frequency enum / stringThis is the frequency of the recurring reminder event (Weekly,Fortnightly,Monthly)
interval numberThis determines the interval of the event ie frequency of weekly and interval of 2 = fornightly
endDate booleanThis is the end date of the recurring reminder event (format YYYYMMDD, end time 'T140000Z')
