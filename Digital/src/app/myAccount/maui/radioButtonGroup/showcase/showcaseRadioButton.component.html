The radio button group allows the user to select the data option from a list of radio buttons.

The group exposes the change event if required. The radio button group also accepts the name and the selected value ie default value

Each radio button takes a label which can be text or HTML (passed in the label section). If passed the radio button also has an accordion section (passed in the body section) that is displayed after the relevant radio button is selected. This section is automatically hidden once another radio button is chosen

Attributes for <agl-maui-radio-button-group>
name string required
selectedValue string
Attributes for <agl-maui-radio-button>
value string required
label string/HTML required
body string/HTML

Refer to MAUI Component Library for details.
