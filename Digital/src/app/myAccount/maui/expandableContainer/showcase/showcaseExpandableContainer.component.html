Expandable container is used to truncate the height of an element when it is deemed too tall (height).
It takes four Input:
  • collapsedHeight: in pixels, the desired height of the element. If the element's height is bigger, the element will be truncated. If the element height is smaller, the expandable container will not influence the layout.
  • gradientHeight: in pixels, the height of the gradient displayed at the bottom when the element is truncated.
  • collapsedMessage: The message to be displayed on the chevron menu when the component is in a collapsed state.
  • expandedMessage: The message to be displayed on the chevron menu when the component is in an expanded state.
