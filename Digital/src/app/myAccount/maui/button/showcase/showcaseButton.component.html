The MAUI Button component consists of:
  • 2 button sizes - Small and Large
  • 3 button types in the MAUI library - Primary, Secondary and Tertiary

It has a default state of a Large Primary Button.

The click event is handled at the parent component.

The event binding used here is `(clicked)` instead of `(click)`.

Attributes
type enum / string (if not set, defaults to 'primary')
size enum / string (if not set, defaults to 'large')
disabled boolean
loader boolean

If no attributes are passed in, button is set to its default state.

Refer to MAUI Component Library for details
