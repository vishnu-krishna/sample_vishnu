A tooltip element that can be used to provide a user with additional information.

The tooltip will shift if its default position would appear out of screen bounds.

The tool tip will be focused on whatever item is placed in the <agl-maui-tooltip> element.

<agl-maui-tooltip-icon> is used to give a 15px circle around a character.

Attributes
tooltipBody string The text in the tooltip bubble required.
position string top(default) or bottom - position determines where the tooltip will be placed relative to the icon.
