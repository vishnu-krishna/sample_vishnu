Segmented Buttons are a graphical representation of an option/radio button, allowing the user to select the data option from a list.

This component has a 2 way binding relationship and has a property [(value)] and event binding (change) which has to be invoked on the host component <agl-maui-segmented-buttons>.

Attributes for <agl-maui-segmented-button>
value string required
text string required
textMobile string (if not set, 'text' will be used for all breakpoints.)
selected boolean required

Refer to MAUI Component Library for details.
