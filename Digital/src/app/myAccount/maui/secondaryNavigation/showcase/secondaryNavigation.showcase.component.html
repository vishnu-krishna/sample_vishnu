Secondary navigation takes four input parameters

Attributes for <agl-maui-secondary-navigation>
text string required
textMobile string (if not set, 'text' will be used for all breakpoints.)
display boolean (false by default)
displayBackButton boolean (false by default)
pageNumber number
pageTotal number
displayPageNumber string 'OnDesktop' | 'OnMobile'
By default, the pageNumber and pageTotal are displayed when both are provided.

Refer to Secondary Nav for details.
