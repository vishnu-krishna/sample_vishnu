The AGL MAUI native dropdown allows the user to select standard options as with any native dropdown. However, the user can also select actions as shown in the example below. After selecting either a function or option from the list, the component returns the value attribute as an identifier.

There are currently no colour formatting options because white is the AGL standard colour for the control. The width of the control is set to 100% and can be configured by sizing the outer container allowing responsive formatting. Actions are deliberately grouped beneath the dashed line in the native dropdown options for usability.

To add placeholder text to a dropdown, add a option that is disabled and selected. See the 'Select a value' option in the example below
