A checkbox element for use in forms and elsewhere.

Transcluded content is allowed in place of label where custom styling can be applied.

If anchor tags are used in transcluded content, event.preventDefault() should be declared in the event binding to prevent checkbox from being selected when link is clicked.

Attributes
label string
value string
isChecked boolean selects the checkbox
isFullWidth boolean allows block to be clickable

There is also a wrapper component to display checkboxes in a group, see Checkbox Group.

Refer to MAUI Component Library for details
