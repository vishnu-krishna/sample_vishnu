Progress tracker is used to demonstrate a timeline and the status of each point on the timeline.
Developer needs to
  • implement "ProgressItem" interface.
  • define tracker dimension css configuration and classes with the mixin helpers in "verticalProgressItemBase.scss" in the component that uses tracker.
  • have ProgressTrackerComponent takes a collection "ProgressItem", iterate and call initialise() on each of them.
  • use "ng-template" to pass "ProgressItem" to "VerticalProgressItemComponent" component.
  • develop pane1 and pane2 components taking data returned from getPane1() and getPane2() of "ProgressItem";
"ProgressItem" interface has 4 functions to implement,
  • initialise(index: number, isLast: boolean)
    • index: the index of the item in collection
    • isLast: is the item the last item in collection
  • getModel() returns progress item model. "VerticalProgressItemModel" class is progress item model for vertical progress tracker
  • getPane1() return pane1 model
  • getPane2() return pane2 model
Extension
  • pane component is not part of the tracker, it can be customized.
  • the component initially support vertical layout, it can be extended to support horizontal layout by adding "horizontalProgressItem"
